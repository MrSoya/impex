x-class支持不同的写法：
  • <div x-class="'cls1 cls2 cls3 ...'>...</div>
  • <div x-class="['cls1','cls2','cls3']">...</div>
  • <div x-class="{cls1:boolExp,cls2:boolExp,cls3:boolExp}">...</div>

style

.c1{ color:red; } .c2{ font-size:20px; } .c3{ font-style: italic; }
